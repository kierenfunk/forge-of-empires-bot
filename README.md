# Forge of Empires Bot
> Bot written in Python for automating various tasks in Forge of Empires

## Information


## Get Started

### Requirements

- Python 3
- `pipenv`

### Installation

With `pipenv`
```bash
pipenv install --dev
pipenv shell
```

## Testing

For linting:
```bash
pipenv run lint
```

To run all tests
```bash
pytest --cov
```
